Backend for small shop site on FastAPI

Options:

    Auth:
      Register user
      Auth user
      Get user info
      Delete user
      Top up money
      Get all orders
  
    Product(CRUD):
      Get one
      Get all
      Buy product
     
    ADMIN:
      Create product
      Update product
      Delete product

Stack: FastAPI, sqlalchemy, alembic, pydantic

link: https://agile-spire-54699.herokuapp.com/docs/
